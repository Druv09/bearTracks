# Bear Tracks - Bridgeland High School Lost & Found

**Description:**
Bear Tracks is a web application designed for Bridgeland High School to manage lost and found items. Students can report lost items, upload photos, search for items, and track claims. Admins can manage submissions, verify claims, and maintain an organized system.

---

## Features

* Student login and admin login with different access levels
* Home page with How-To guide for first-time users
* Submit Lost Item page with photo uploads and detailed reporting
* Found Items/Search page with filtering by category, location, and date
* Admin Dashboard showing total users, pending claims, and submitted items
* Admins can approve, delete, or clear pending claims

---

## Technology Stack

* **Frontend:** React
* **Deployment:** Netlify
* **Styling:** CSS (with school colors: Orange & Blue)

---

## Installation

1. Clone the repository: `git clone https://github.com/yourusername/beartracks.git`
2. Navigate to the project folder: `cd beartracks`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
5. Open `http://localhost:3000` in your browser

---

## Usage

1. Login as a student to submit lost items, search items, or claim items.
2. Login as an admin to manage users, view pending claims, approve or delete items.
3. Use the Home page How-To guide for navigation instructions.

---

## Page Descriptions

* **Home Page:** Shows a welcome message and How-To guide explaining how to report or claim items. Quick links to the Search and Submit Lost Item pages are provided.
* **Admin Dashboard:** Displays total users, total items, and pending claims. Admins can approve, delete, or manage items and even clear all pending claims at once.
* **Login Page:** Separate login options for students and admins, each with their own permissions and access levels.
* **Found Items/Search Page:** Users can browse all found items and filter by category, location, or date. Photos and descriptions help identify items.
* **Submit Lost Item Page:** Students can report a lost item, upload photos, and provide details like title, description, date, and location.
